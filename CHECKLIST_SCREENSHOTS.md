# Checklist Screenshot untuk Panduan Admin

Karena keterbatasan sistem untuk mengambil screenshot otomatis, mohon bantuan Anda untuk mengambil screenshot berikut dan menyimpannya di folder `docs/img/`.

Pastikan aplikasi berjalan (`php artisan serve`).

| No | Halaman / Aksi | URL | Nama File Gambar (Simpan di `docs/img/`) |
|----|---------------|-----|-------------------------------------------|
| 1. | Tampilan Layar Utama | `http://localhost:8000/` | `display_home.png` |
| 2. | Halaman Login | `http://localhost:8000/admin/login` | `login_page.png` |
| 3. | Sidebar Menu (Sorot menu Jadwal) | `http://localhost:8000/admin` | `jadwal_menu.png` |
| 4. | Form Tambah Jadwal | Klik Jadwal -> Tambah | `add_schedule.png` |
| 5. | Form Tambah Video | Klik Video -> Tambah | `add_video.png` |
| 6. | Form Tambah Carousel | Klik Carousel -> Tambah | `add_carousel.png` |
| 7. | Form Tambah Background | Klik Background -> Tambah | `add_background.png` |
| 8. | Halaman Settings | Klik Settings | `settings.png` |
| 9. | Penjelasan Layout (Edit `display_home.png` dengan kotak-kotak penjelasan jika mau, atau biarkan kosong) | `display_home.png` | `display_layout_explanation.png` |

## Cara Mengambil Screenshot (Windows):
1. Buka halaman yang diinginkan.
2. Tekan `Windows + Shift + S`.
3. Pilih area yang mau diambil.
4. Simpan gambar ke folder `c:\laragon\www\test\display_informasi_v2\docs\img\`.
5. Beri nama sesuai kolom "Nama File Gambar".

Setelah semua gambar ada, file `PANDUAN_ADMIN.md` akan otomatis menampilkan gambar-gambar tersebut.
